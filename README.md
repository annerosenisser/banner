# banner
Replication files for Nisser &amp; Weidmann. "Measuring Ethnic Preferences in Bosnia and Herzegovina with Mobile Advertising." PLoS One.

To run the analysis, click on the upper right green buttom "Clone or download", download as zip file, and then open the file "banner.Rproj" in RStudio. This opens the entire project. In this way, you have directly set the right working directory, including the right subdirectories for accessing the data.
